# Crypto Dashboard – Full Stack Project

Dashboard de monitoramento de criptomoedas em tempo real.

## Tecnologias
- Frontend: React + TailwindCSS
- Backend: Node.js + Express
- APIs externas: CoinGecko (preços), NewsAPI (notícias)

## Funcionalidades
- Consulta de preços de criptomoedas com gráficos.
- Notícias em tempo real.
- Dark mode, filtros, busca, alternância cards/tabela.
- Animações inteligentes via Intersection Observer.

## Como rodar
1. `npm install` no frontend e backend
2. Criar `.env` com chave da NewsAPI
3. `npm start` no backend
4. `npm start` no frontend
